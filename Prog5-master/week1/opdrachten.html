In deze opdracht ga je een Console Applicatie maken in het .NET Framework met behulp van Visual Studio en C#. C# lijkt qua syntax heel erg op Java dus het is niet erg als je nog nooit eerder iets met C# gedaan hebt, daar waar nodig worden hints gegeven.

Een nieuwe taal leren? Dan eerst hello world! We gaan het alleen iets moeilijker maken. Bouw een console applicatie waar de console vraagt om een naam. Nadat de gebruiker een naam heeft ingevoerd moet de console "hello" + naam printen.

  • Let op dat de template die je kiest een console application template is.

Voor deze opdracht is er een startupproject beschikbaar. In dit project zitten interfaces waar je gebruik van kunt maken en een test die je kunt runnen. Het is niet verplicht om gebruik te maken van deze startup, dus je kunt ook zelf een nieuw project beginnen.

Het doel van deze oefening is een eerste kennismaking met C# en het leren omgaan met Visual Studio. Maak in Visual Studio de Console Application template een C# programma die een geboortedatum van een persoon vraagt en vervolgens de leeftijd van deze persoon berekent en toont. De applicatie dient aan de volgende voorwaarden te voldoen:

  • De datum dient via de Console opgevraagd te worden en aan het format (dd-mm-yyyy) te voldoen.
  • Als een ongeldige datum of ongeldige input gegeven wordt dient dit op de Console aangegeven te worden.
  • Bij een geldige datum wordt de leeftijd berekend en getoond.

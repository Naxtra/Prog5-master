Stappenplan

Stap 1- forken

Om een pull request uit te voeren is het makkelijk om niet meteen de repository te klonen, maar eerst een 'fork' te maken.

De gemaakte fork is daarna de een online repository waar je zelf in gaat werken.

Stap 2 - Lokaal klonen

Kloon deze (de fork) repository lokaal in plaats van de repository Avans/Prog5.

In deze lokale versie kun je nu de verbeteringen aanbrengen. Vergeet deze verbeteringen (lokaal) niet te pushen naar je eigen remote repository! (de fork)

Als alle changes compleet op je fork repository staan ben je klaar om je pull request uit te voeren!

Stap 3 - De pull request

Als het klopt zie je in je fork iets vergelijkbaars met het onderstaande

Als je nu op 'pull request' klikt zullen alle changes in je fork doorgevoerd worden in de repository van Avans/Prog5.

Nadat ik (Stijn) deze changes goedkeur, worden ze automatisch weergegeven op het webdictaat.

Alvast bedankt voor de inspanning!
